Bucket policy
The bucket policy, written in JSON, provides access to the objects stored in the bucket. 
Bucket policies don't apply to objects owned by other accounts. 
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:GetObject"
            ],
            "Resource": [
                "arn:aws:s3:::Bucket-Name/*"
            ]
        }
    ]
}
